# Ellty Test – Checkbox List Component

Pixel-perfect replication of the Figma component: a small card with a list of pages and a yellow **Done** button.

## Live locally
Open `index.html` in any modern browser.

## Deploy
- GitHub Pages: push these files to a public repo → Settings → Pages.
- Vercel/Netlify: drag & drop or import the repo.

## Tech
- HTML + CSS + vanilla JS
- Accessible: semantic roles, focus styles, keyboard-friendly
- Accurate UI: spacing, radii, shadows, hover/active states
